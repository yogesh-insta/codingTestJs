## Arithmetic Series

### With first and last number in the series. 
`Sum = n/2 * (a1 + an)`
### With first number and common difference 
`Sum = n/2 * [2a1 + (n-1)d]`

- a1 - first term
- an - nth term
- n - number of terms
- d - common difference
